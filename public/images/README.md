# Imagens do Site

## Como adicionar as imagens:

### 1. Logo (logo-venda-forte.png)
- Salve a logo circular vermelha com o "VF" branco como: `logo-venda-forte.png`
- Formato recomendado: PNG transparente
- Tamanho: 200x200px ou maior

### 2. Prédio/Sede (prediu-venda-forte.jpg)
- Salve a foto do prédio da empresa como: `predio-venda-forte.jpg`
- Formato: JPG ou PNG
- Tamanho recomendado: 1200x800px

### 3. Como adicionar:
1. Salve as imagens que você tem
2. Renomeie para os nomes acima
3. Cole nesta pasta: `public/images/`
